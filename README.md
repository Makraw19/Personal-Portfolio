# MSA Portfolio

Personal portfolio for MD Sheikh Abdullah — Sr. QA Engineer, Sydney.

## Deploy

1. Upload all files in this folder to a GitHub repo
2. Settings → Pages → Source: Deploy from a branch → main → / (root) → Save
3. Wait ~1 minute — site goes live at https://USERNAME.github.io/REPONAME/

- `case-study.html` — PenCS Practice Cloud case study
